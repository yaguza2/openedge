Provides a set of tools that makes working with the OpenEdge Access framework easier.

The UserTool is a command-line interface on an implementation of UserManager. It uses 'oeaccess.xml' in this package as the default configuration document. You can overide this location by setting the environment variable 'oeaccess.configFile' to the correct url.

usage: usertool
 -addrole          adds a role (for a user if given)
 -adduser          adds a user
 -delrole          deletes a role completely
 -deluser          deletes a user
 -help             prints this message
 -listroles        lists roles (for a user if given)
 -listusers        lists users (for a role if given)
 -p,--pwd     the password of the user
 -r,--role    the name of the role
 -removerole       removes a role for a user
 -resetpwd         changes the password of a user
 -u,--user    the username of the user
@see nl.openedge.access.tools.UserTool @see nl.openedge.access.UserManager
