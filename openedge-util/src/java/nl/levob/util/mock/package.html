Bevat mock classes om testen met mockobjects makkelijker te maken.

Mock classes voor testen van Baritus.

  • AbstractMockControlTest.
  • MockHttpServletRequest. Mock voor HttpServletRequest.
  • MockHttpServletResponse. Mock voor HttpServletResponse.
  • AbstractBaritusTestCtrl. Abstracte klasse voor testen van Baritus controls, waarbij deze klasse als basis kan dienen voor de te testen controls. AbstractBaritusTestCtrl maakt veel van de normaliter protected methoden public, zodat we er makkelijk bij kunnen van buitenaf. Zo kunnen we bijvoorbeeld validators registreren voor een testcontrol zonder dat we dat steeds in de control zelf hoeven te doen (we zouden anders een aanzienlijke hoeveelheid controls krijgen).
