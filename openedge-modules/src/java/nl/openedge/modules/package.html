This package has the main classes and interfaces for the Open Edge Module framework.

The OpenEdge module framework provides for an easy to use lightweight container for components and Jobs.

Use the JDOMConfigurator to initialize the framework from an XML file/ stream.
The RepositoryFactory can be used to get the current repository.
You can then use the current ComponentRepository to get instances of components.
The AbstractComponentFactory functions as a common base class for implementations of ComponentRepository
