This package contains the loosely typed implementation of the ComponentRepository and TypesRegistry that is used by it. The LooselyTypedComponentRepository regconizes types by the configuration of the component.

If you do not want to have dependencies on interfaces of this framework, this is the way to go.

Initialize like this:
RepositoryFactory.setImplementingClass(LooselyTypedComponentRepository.class.getName()); JDOMConfigurator c = new JDOMConfigurator(url);
